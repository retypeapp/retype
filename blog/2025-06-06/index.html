# GitHub Pages setup

With the community key, you can now setup your project to be hosted using GitHub Pages. Please review the documentation to configure a GitHub Pages project with Retype with automatic deployment using GitHub Actions.

Do you have any questions? Feel free to contact us using the live chat (see bottom right corner). If we're online, we will be more than happy to answer questions or step you through any setup troublshooting.

# What is included

  • Open-source and Commercial use
  • Unlock all projects hosted on *.github.io
  • Up to 1000 pages
  • Free, no cost
  • Free Retype Pro upgrades
  • Retype Pro features, including nav, scheme, outbound, breadcrumb, plus all future Retype Pro features

If your project is hosted using GitHub Pages with the default github.io domain name, the GitHub Pages community key is available for you.

# What is not included

  • Powered by Retype branding is not removed
